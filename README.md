A comprehensive, interactive 3-page Power BI dashboard analyzing Netflix's global library. This project explores content distribution, production trends, and regional footprints with a high-end cinematic UI.

---

### 🚀 Key Dashboard Features
This report consists of three dedicated analytical views with seamless button navigation:

#### 1. Global Dashboard (Overview)
- **KPIs:** Tracked Total Platform Content (4,808), Movies (4,649), and TV Shows (159).
- **Production Trends:** A growth trend analysis showing a massive surge in content production peaking around 2020.
- **Content Mix:** Visual representation of the dominance of Movies (96.7%) over TV Shows.

#### 2. Movies Deep Dive
- **Runtimes:** Analyzed the average movie duration (101 mins) vs. the longest duration (253 mins).
- **Genre Insights:** Top 10 genres identified, with **Dramas** and **Comedies** leading the catalog.
- **Drill-Down:** Release year catalog for detailed title-level exploration.

#### 3. TV Shows & Global Footprint
- **Geospatial Mapping:** Global volume by country using interactive map visuals.
- **Top Nations:** The United States and UK identified as the primary content hubs.
- **Binge Trends:** Analysis of longest-running TV shows, highlighting titles like *Supernatural* (15 Seasons).

---

### 🛠️ Technical Skills Applied
- **UI/UX Design:** Custom branding using a "Netflix Red" theme for a professional user experience.
- **Data Modeling:** Integrated page navigation using custom buttons and bookmarks.
- **DAX & Power Query:** Data cleaning and metric calculations (Average durations, title counts).
- **Visualization:** Donut charts, Area charts, Map visuals, and Bar charts.

---

### 📂 How to View
1. Download the `.pbix` file from this repository.
2. Open with **Power BI Desktop**.
3. Use the **Navigation Buttons** at the bottom to switch between views.
